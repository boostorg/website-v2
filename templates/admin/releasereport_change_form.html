Publishing:
  • Report PDFs are not archived or versioned. Publishing an older version is not possible after a newer one has been published.
  • For any Report Configuration it's only possible to republish the most recently published report, or a new draft.
  • To publish a draft, for safety, all existing entries for that Report Configuration must be manually deleted first.
