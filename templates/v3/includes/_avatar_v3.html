{% comment %} Avatar (v3) Displays a user avatar as an image, initials, or placeholder. Variables: - src (optional) Image URL for the avatar - name (optional) User's full name; used for alt text and initials fallback - variant (optional) Background color: yellow | green | teal (default: yellow) - size (optional) Avatar size: sm | md | lg | xl (default: md) Rendering priority: 1. Image — when `src` is provided 2. Initials — when `name` is provided (falls back to "?" if initials can't be derived) 3. Placeholder — generic "?" icon when neither `src` nor `name` is given Accessibility: - When `name` is provided, the avatar is announced to screen readers via aria-label. - When `name` is absent, the avatar is treated as decorative (aria-hidden="true"). - Always pass `name` alongside `src` so the image has meaningful alt text.
